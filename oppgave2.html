Media Typer

Media typer blir brukt for tilpassing for teknologi og tilpassing til menneske. Det er mykje under CSS Media typer, men dei har som regel ei rolle og det er for å gi meir brukarvennlighet og gi ulike datamaskiner tilgang til nettsida di. grunnen til kvifor ein play station portable kan lese html filer er som regel på grunn av at noken har lagt inn media typer som den datamaskinen kan skalere ned på og gjære det lesebart for ein bruker. Det er også media typer som har sin eigen unike funksjon. for eksempel så er media typen braille intendert for folk med nedsatt syn, men det er ikkje slik at alle mediatypane er fortsatt i bruk. mediatypen handheld for eksempel, blir nesten alldri brukt på grunn av skalering og responsiv design. den spessfikasjonen vart brukt for mobiltelefonar og liknande og poenget med den var for å lage ei god nettside for folk som brukte små datamaskiner som mobiltelefonar. mediatyper har meir allsidigheit enn det. for eksempel så kan mediatypene print og speech, printe ut ein html dokument og speech kan gjære den om til lyd. mediatyper har vore i webutvikling i ein god stund no, så kva er fordelene og ulempene? og er det mediatyper som held på å miste sin bruk?

Fordeler

det er veldig mange gode fordeler for media typer. denne nettsida her har media typer i seg. ei media type som heiter @media screen for eksempel. oppgåva til den mediatypen er å skalere ei nettside slik at den blir leselig frå ein liten mobiltelefon til ein 4K skjerm. ein anna fordel med media typer er at personar med netsatte evner som syn kan få tilgang til ei nettside. braille og embossed er to gode eksempler på det. braille er laga for folk med nedsatt syn, og embossed er intendert for å skrive ut dokumenter for personar med nedsatt syn. embossed skriver ut tekst i blindskrift. print er intendert for documenter som er intenderte for bli skrevet ut. det er mange media typer og alle har sin grunn, men den teknologiske verda utvikler seg dagleg, og med det kommer det nye måtar å lage nettsider på som gjer ulike media typer ikkje så effektive. Stort sett er det eit fordel med tanke på at denne nye måten å skrive på er som regel ein forbedring av det som var tidlegare.

Ulemeper

Ulemper med media typer er at med utvikling så kommer det utdaterte media typer. dei virker til ein viss grad, men ikkje alltid og det krever endring for ulike nettsider for at dei skal bli moderne. media typen projection begynner å bli utdatert på grunn av responsiv design (media queries). gamle nettsider bytter ut strukturen i heile nettsida si for å bli moderne. Dette er ei ulempe, men med slik fart teknologi utviklar seg på er det normalt med slike endringar.

Forbruk av Media Typer

Som sagt så er det fleire media typer som blir utdaterte etter kvart på grunn av teknologisk utvikling. responsiv design har gjort fleire mediatyper unødvendige. handheld og projection er berre nokre av media typane som har blitt unødvendige på grunn av responsiv design. det går fortsatt ann å bruke dei det er berre ikkje effektivt når du kan like så godt berre skrive responsivt koden berre gjør jobben for alt innanfor dette.
